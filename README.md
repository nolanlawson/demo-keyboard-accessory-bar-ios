Broken Keyboard.hideFormAccessoryBar
============ 

Demo of broken  Keyboard.hideFormAccessoryBar in iOS 8.

Building and running
-------

```bash
# check out this repo and the keyboard plugin
git clone https://github.com/nolanlawson/demo-keyboard-accessory-bar-ios.git
git clone https://github.com/apache/cordova-plugins.git

# add the plugin
cd demo-keyboard-accessory-bar-ios
cordova plugin add ../cordova-plugins/keyboard

# run on iOS
cordova platform add ios
cordova run ios
```
